# VideoCap_ClipExtract

## Run the project

Step # 01
  open cmd in projects's folder

Step # 02 (hit this command in cmd)
  uvicorn api:app --reload
  
Step # 02
  it will open the fastapi's swagger with default address http://127.0.0.1:8000
